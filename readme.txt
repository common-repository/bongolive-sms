Send SMS via Bongolive services in wordpress

== Description ==
This plugin lets you communicate with all of your customers and inviduals who visit your website or blog via mobile sms using <strong>Bongolive SMS services</strong>.

Bongo Live is a unique online platform to manage sms campaigns and communication. 
Visit our <a href="http://bongolive.co.tz">website</a> to learn about our complete range of services or <a href="http://bongolive.co.tz/contactus.php">Contact Us</a> for more details.

To setup: The plugin user must have a vendor account with Bongo Live. To create an account visit <a href="http://bongolive.co.tz/">Bongolive</a> and Sign-up as a vendor/broadcaster.
Once complete follow instructions on the web page on how to purchase credits and get more sender names. Revisit your plugin and start sending SMS to all of your subscribers and other people.
Contact us for international sms support.

Note: The plugin sends requests to Bongolive API services to retrieve user credit balance and sender IDs. It then sends SMS with message content to page subscribers/web-visitors via Bongo Live API services.

P.S: You need your username, password and APIkey to use this plugin, get it from your Bongo Live broadcaster profile. View screenshots for more details.

Features:

* Send SMS to a single number or multiple numbers at once.
* Send SMS to subscribers
* Create groups for subscribers to join
* Send SMS to one or multiple groups
* Widget support for sigining-up subscribers
* Widget takes theme appearance settings
* Wordpress shortcode support to add subscription box to any page -  "[bongolive]"
* Send post title and post link to subscribers automatically via sms
* Send activation sms to subscribers when subscribing.


Language Support:

* English

== Installation ==
1. Download the plugin from the wordpress plugin directory or
1. Upload `bongolive-sms` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To display Subscribe goto Themes -> Widgets, and adding `Subscribe to SMS` into your sidebar or using this Shortcode `[bongolive]` in Posts pages or Widget.
